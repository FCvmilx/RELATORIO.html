Relatório da Avaliação 3

Introdução

Este relatório consiste em apresentar a documentação referente à Avaliação 3. O objetivo do terceiro trabalho é a implementação de uma biblioteca para o Tipo Abstrato de Dado StringSet utilizando a estrutura TRIE. Será exibido no relatório a explicação de cada função implementada no código (escrito na linguagem C) e exemplos de execução no fim.

Definição de Dados

                

                    #define MAX 27
                    #define AVISO(msg) fprintf(stdout, "AVISO: %s\n", #msg)

                    typedef char *String;

                    typedef enum {
                        false,
                        true
                    }bool;

                    typedef struct no {
                        char dado;
                        struct no* filhos[MAX];
                        bool fim;
                        String conteudo;
                    }No;

Para este programa, utilizamos a estrutura de TRIE TREE. Para criar esta estrutura, utilizamos as definições acima.

A linha #define MAX 27, define uma constante global que será usada para definir o conjunto de possibilidades de caracteres da entrada, sendo as 26 letras minúsculas do alfabeto e o terminador de string : ‘\0’.

Em seguida, definimos que o vetor de caracteres se chamaria String. Após isso, definimos um tipo de dado booleano.

E por fim, definimos a estrutura geral do nó da árvore.

Funções

  1. criaNo

    2. No* criaNo(char Letra)

    Retorna um nó alocado dinamicamente.
  2. liberaNo

    3. void liberaNo(No* node)

    Desaloca a porção de memória alocada para o Nó.
  3. Insere

    4. void Insere(No* raiz, String palavra)

    A função de inserção recebe uma palavra e adiciona cada caractere na árvore, de forma a criar novos nós caso não exista na árvore.
  4. Pertence

    5. bool Pertence(No* raiz, String palavra)

    Função que percorre a árvore buscando o índice de cada caractere da string buscada. Ao chegar na última letra da palavra, verifica se o nó é folha e não é nulo.
  5. Vazia

    6. bool Vazia(No* raiz)

    A função verifica se a variável global que contabiliza o número de palavras é igual a 0.
  6. Remove

    7. No* Remove(No* raiz, String palavra, int index)

    Remove uma palavra da árvore: A função percorre todo o caminho que leva até a folha e remove o nó apenas se este não tiver filhos.
  7. Imprime

    8. void Imprime(No* raiz, String palavra , int index)

    Percorre toda a árvore imprimindo as palavras. Ao encontrar um nó folha percorre seguindo a ordem de índices que basicamente deixa a árvore lexicograficamente ordenada.
  8. ImprimeComPre

    9. void ImprimeComPre(No* raiz, String palavra, String prefix, int index)

    Similar à função de impressão, porém recebe um nó interno que é o fim do prefixo, além de receber a string prefixo para imprimi-la junto aos nós subsequentes.
  9. Prefixo

    10. void Prefixo(No* raiz, String prefixo)

    Responsável por percorrer o caminho do prefixo na árvore e chamar a função ImprimeComPre() para realizar a impressão dos Nós.
  10. Tamanho

    11. int Tamanho()

    Retorna o valor da variável global que contabiliza o número de palavras adicionadas.
  11. CTOI

    12. int CTOI(char letra)

    recebe um caractere e retorna seu valor inteiro na tabela ASCII.
  12. apagaArvore

    13. No* apagaArvore(No* raiz)

    Recebe o nó raiz e percorre toda árvore até chegar em um Nó externo, que, caso não tenha filhos, é removido.
